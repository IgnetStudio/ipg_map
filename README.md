## Installation

1. Install [node.js](https://nodejs.org/en/)

2. Clone repository

`git clone https://github.com/IgnetStudio/ipg_map.git`

3. Install dependencies:

`yarn`

4. Launch page locally (http://localhost:3000/):

`yarn run dev`
